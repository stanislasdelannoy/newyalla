# Yalla

### Helpers

##### Init un repo

```{bash}
mkdir <repo>
cd <repo>
git init
# Créer le repo sur github
git remote add origin <lien_du_repo>
touch README.md
git add .
git commit -m
git push origin master
```

##### Créer une app React from scratch

```{bash}
npx create-react-app <nom-app>
```
